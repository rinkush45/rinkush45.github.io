# 🖥️ RINKU_SHARMA // NEURAL_OS v2.0.4

> **System Architect & DevOps Engineer Portfolio**  
> A high-fidelity, interactive terminal-inspired operating system interface built with Next.js 15.

---

## 🌌 Project Overview

**NEURAL_OS** is a premium, pixel-perfect portfolio experience designed to reflect the technical depth of a Systems Architect. It eschews traditional layouts for a cyber-minimalist "Command Center" aesthetic, featuring glassmorphism, real-time system visualization, and deep terminal immersion.

---

## ⚡ Key Features

### 📡 Real-time Systems Telemetry
The sidebar communicates live "OS Health" metrics:
- **CPU Load**: Dynamic orchestration simulation.
- **RAM Usage**: Real-time memory allocation monitoring.
- **Network Traffic**: Live throughput visualization.
- **Active Users**: Real-time session monitoring.
- **System Clock**: Precise atomic time synchronization.

### ⌨️ Advanced Terminal Dynamics
- **Typewriter Engine**: A custom-built asynchronous typing system that cycles through core competencies with randomized human-like velocity.
- **Matrix Rain Canvas**: A high-performance HTML5 Canvas implementation of the iconic "Digital Rain," customized with personalized character sets.
- **Glitch UI**: Subtle CSS-based glitch effects and scanline overlays to enhance the "analog-future" atmosphere.

### 📂 Strategic Projects Archive
- **Lazy-Loaded Grid**: Optimized project cards with individual impact visualizations.
- **Accordion Expansion**: Detailed case studies that "wipe-in" horizontally, revealing technical problems, solutions, and measurable impact.
- **Full Architecture Archive**: A dedicated subsystem (`/projects`) for exploring the complete history of engineered solutions.

## 🚀 Getting Started

### 1. Initialize Connection
Clone the repository to your local terminal:
```bash
git clone https://github.com/rinkush45/Portfolio_v2.git
cd Portfolio_v2
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Launch Development Server
```bash
npm run dev
```
Navigate to `http://localhost:3000` to witness the boot sequence.
